## ðŸ›  DevOps Branching Exercise

Since you are learning branching, follow this workflow in your terminal/VS Code:

1. **Initialize Git**: `git init`
2. **Create a Feature Branch**: `git checkout -b feature/add-sell-tab`
3. **Check out the files**: Look at `pages/Sell.tsx` to see how the new tab works.
4. **Practice a Change**: Modify the "Condition" options in `pages/Sell.tsx`.
5. **Commit**: `git add .` and `git commit -m "feat: updated selling form conditions"`
6. **Merge**: `git checkout main` and then `git merge feature/add-sell-tab`
